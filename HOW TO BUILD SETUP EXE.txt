HOW TO GET YOUR "Dev Draft POS Setup.exe"
==========================================

This guide gets you a professional Windows installer EXE
without typing any commands. Takes about 10 minutes total.

WHAT YOU NEED:
- A free GitHub account (github.com)
- Internet connection
- That's it.

─────────────────────────────────────────────────────────
STEP 1 — Create a free GitHub account (skip if you have one)
─────────────────────────────────────────────────────────
1. Open browser, go to: https://github.com
2. Click "Sign up"
3. Enter your email, create password, username
4. Verify your email

─────────────────────────────────────────────────────────
STEP 2 — Create a new repository
─────────────────────────────────────────────────────────
1. After logging in, click the "+" button (top right)
2. Click "New repository"
3. Name it: devdraft-pos
4. Make it PRIVATE (important - your code is private)
5. Do NOT check any checkboxes
6. Click "Create repository"

─────────────────────────────────────────────────────────
STEP 3 — Upload your code
─────────────────────────────────────────────────────────
1. On the new repository page, click "uploading an existing file"
2. Drag and drop ALL files from this folder into the upload box
   (select all files and folders — include .github folder)
3. Scroll down, click "Commit changes"
4. Wait for upload to finish

─────────────────────────────────────────────────────────
STEP 4 — Watch it build automatically
─────────────────────────────────────────────────────────
1. Click the "Actions" tab at the top of your repository
2. You will see "Build Dev Draft POS — Windows Installer" running
3. There will be an orange circle (running) — wait for it
4. It takes about 5-10 minutes
5. When done, it turns GREEN ✓

─────────────────────────────────────────────────────────
STEP 5 — Download your Setup.exe
─────────────────────────────────────────────────────────
1. Click on the green build run
2. Scroll down to "Artifacts" section
3. Click "Dev-Draft-POS-Setup" to download
4. Extract the zip — inside is "Dev Draft POS Setup.exe"
5. Double-click it to install your app!

─────────────────────────────────────────────────────────
WHAT THE INSTALLER DOES:
─────────────────────────────────────────────────────────
✓ Shows welcome screen with your app name
✓ Shows license agreement
✓ Lets user choose install folder
✓ Installs app to C:\Program Files\Dev Draft POS\
✓ Creates Desktop shortcut automatically
✓ Creates Start Menu shortcut
✓ Option to launch app immediately after install
✓ Proper Uninstaller in Windows Settings > Apps

─────────────────────────────────────────────────────────
NEED TO UPDATE THE APP LATER?
─────────────────────────────────────────────────────────
1. Upload your changed files to GitHub (same repository)
2. GitHub builds a new Setup.exe automatically
3. Download and share the new installer
