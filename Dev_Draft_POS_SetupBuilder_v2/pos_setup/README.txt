HOW TO RUN — CHOOSE YOUR METHOD:
═══════════════════════════════════

METHOD 1 — INSTANT LAUNCH (No installation needed)
────────────────────────────────────────────────────
• Double-click: LAUNCH_APP_NOW.vbs
• Opens in a dedicated Edge/Chrome app window (frameless)
• Works on any Windows PC with Edge (comes with Windows 10/11)
• Data saved to: renderer\data\posdata.json
• Excel exports: renderer\Excel_Exports\

METHOD 2 — BUILD TRUE WINDOWS EXE (Recommended for production)
───────────────────────────────────────────────────────────────
• Install Node.js from https://nodejs.org/ (LTS version)
• Double-click: BUILD_EXE_WINDOWS.bat
• Wait for the build to complete (~5 minutes, requires internet)
• Find your EXE at: dist\Dev Draft POS.exe
• Double-click the EXE to run — no browser needed!
